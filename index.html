# UX Research # UX Designer

我擁有 2 年以上的 UI/UX 專案經驗，擅長從初始概念到最終交付的完整設計流程，並能將複雜需求轉化為直覺且好用的解決方案。設計對我來說不只是把畫面做漂亮，而是幫使用者找到最自然、最舒服的體驗！

我的背景為人機互動設計，專長於使用者研究、以人為中心的設計方法與體驗設計，專注於創造貼近人性需求、又能真正落地實現的解決方案。

{ ADVANTAGES }

注重細節與使用者感受

能從使用者的語言、行為與情境中理解深層需求，透過敏度觀察進行方法歸納隱藏的痛點，讓設計決策更貼近使用者的核心價值。

問題定義能力

將複雜的挑戰轉化為清楚的設計問題，幫助團隊聚焦並找到解決方法。

視覺化與溝通

能將研究結果整理總結，轉化成易於理解的圖像與流程圖，促進團隊協作。

情境導向與可及性

從不同使用情境思考多樣化的使用者條件出發，讓設計在更多情境下都能被理解與使用。

跨團隊協作能力

具備 HTML 與 CSS 的理解，能以設計語言與技術語言雙向溝通，減少設計與開發間的落差。

敘事化設計能力

能把研究發現與洞察轉化成有故事/情境性的呈現，幫助團隊"感受到"使用者的需求，進而推動設計方向。
